# IBS Landing Page

A clean, responsive landing page for IBS awareness and lifestyle guidance, emphasizing layout, readability, and subtle animations. This project focuses on front-end structure, design consistency, and basic animation integrations.

https://ibs-landing.netlify.app/

## Features

Responsive Layout – Mobile-first design using CSS Flexbox and Grid.

Hero Section – Visually appealing introduction with call-to-action.

Card Section – Highlights symptoms, diet plans, and treatments.

Split Sections – Alternating image/text sections for readability.

Subtle Animations – Smooth hover effects, fade-ins, and scroll-based transitions.

Accessible Typography – Clear font hierarchy for easy reading.

## Technologies
HTML5 – Semantic structure for accessibility and SEO.

CSS3 – Styling, layout, and animations.

Google Fonts – Clean, readable typography.
